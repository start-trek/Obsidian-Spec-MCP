# QuickAdd Spec Pack

Sources
- QuickAdd Docs: https://quickadd.obsidian.guide/docs/
- QuickAdd VALUE docs: https://publish.obsidian.md/quickadd/VALUE

Purpose
- Capture the formatting tokens and choice-oriented authoring patterns used by QuickAdd.

Rules
- QuickAdd format tokens generally use double braces.
- Common built-ins include `{{DATE}}`.
- Variable capture can use `{{VALUE:name}}`.
- Keep placeholders balanced and compatible with the specific QuickAdd choice configuration in your vault.

Examples
- `{{DATE}}`
- `{{VALUE:project}}`
