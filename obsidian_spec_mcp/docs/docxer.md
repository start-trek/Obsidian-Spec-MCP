# Docxer Spec Pack

Sources
- Docxer GitHub: https://github.com/Developer-Mike/obsidian-docxer

Purpose
- Model the workflow for converting `.docx` files into markdown inside Obsidian.

Rules
- Docxer is workflow-oriented rather than a standalone markdown grammar.
- Track the source `.docx` and the resulting markdown output note.
- Keep generated content linter-friendly after conversion.

Example workflow
1. Add a `.docx` file to the vault.
2. Open it in Obsidian.
3. Convert it to markdown.
